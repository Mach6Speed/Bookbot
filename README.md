# BookBot

BookBot is a small Python command-line app that analyzes a text file and prints:

Total word count
Character frequency counts (letters only)
A formatted report in descending character-count order

It is useful for quick text analysis of novels or any plain-text document.

## Requirements

- Python 3 installed

## How To Use

Open a terminal in this project folder.
Run the script with a path to a text file.

## Example Input Files

This repository includes sample books in the `books` folder:

- `books/frankenstein.txt`
- `books/mobydick.txt`
- `books/prideandprejudice.txt`

## What The Output Looks Like

The report will print as follows:

- BOOKBOT
- Word Count
- Sorted Character Count
- END

In the Character Count section, each line looks like:

- a: 12345
- b: 12233
- c: 12000
